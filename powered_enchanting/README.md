Update from older Versions<br>
If you are currently using an old version of the Data Pack and want to update to the new one, you need to install the Compatibility Data Pack.<br>
This data pack updates all your tools to be compatible with the new system.<br>
The update of the tool is done when it is in the inventory of a player. When you are sure that all items with custom enchantments have been updated, you can remove the pack.


Update custom Addon's
In give.mcfunction you need to replace Powerench with PoweredEnchanting:<br>
`give @s minecraft:enchanted_book{Powerench:[{id:"minecraft:`<br>
to<br>
`give @s minecraft:enchanted_book{PoweredEnchanting:[{id:"minecraft:`
